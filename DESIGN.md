## 2) التخطيط (Layout)
- **الشريط الجانبي:** عرض ثابت `248px`، خلفية بيضاء، حد على الحافة الداخلية (`border-e`). يحوي: العلامة (مربّع teal + اسم + وصف) ← عنوان مجموعة بأحرف صغيرة خافتة («مساحة عمل الوسيط») ← عناصر التنقل ← مجموعة «الإعدادات» ← تذييل **وضع تجريبي** (نقطة برتقالية).
- **الشريط العلوي:** ارتفاع `64px`، لاصق أعلى، حد سفلي. يحوي: بحث ممتد (دائري) ← مبدّل اللغة (AR/EN + كرة أرضية) ← جرس إشعارات بنقطة حمراء ← بطاقة المستخدم (حرف أولي + اسم + دور).
- **المحتوى:** حشوة `~24px`، شبكة بطاقات KPI (1/2/4 أعمدة حسب العرض)، ثم أقسام.
- **RTL/LTR:** يُضبط `dir` على `<html>` حسب اللغة؛ استخدم الخصائص المنطقية (`ps/pe`, `ms/me`, `start/end`, `border-e`) لا اليمين/اليسار الصريحة. الأيقونات الاتجاهية تُقلب بـ `rtl:rotate-180`.

## 3) المكوّنات
- **عنصر تنقل نشط:** خلفية `--primary-soft`، نص `--primary-strong`، وزن semibold. غير النشط: نص `--text-muted` مع hover خفيف.
- **عنصر «قريباً»:** غير قابل للنقر، نص خافت + شارة صغيرة رمادية «قريباً».
- **بطاقة KPI:** بيضاء، زوايا `14px`، ظل خفيف، حد؛ عنوان خافت صغير + قيمة كبيرة (24px bold) + سطر فرعي خافت + رقاقة أيقونة ملوّنة في الحافة الخارجية.
- **شارة حالة (Badge):** دائرية كاملة، `px-2 py-0.5`، نص 11.5px، خلفية ناعمة + نص ملوّن حسب الحالة.
- **زر أساسي:** `--primary-strong` خلفية، نص أبيض، زوايا `8–10px`. **زر ثانوي:** أبيض بحدّ، نص خافت، أيقونة (مثل «تصدير»).
- **التبويبات:** مجموعة على مسار `--surface-2`؛ النشط بطاقة بيضاء بظل + عدّاد في شارة.
- **الجداول:** رؤوس أعمدة بأحرف صغيرة خافتة (11px)، فواصل صفوف خفيفة، hover خفيف، أرقام بـ `tabular-nums` (صنف `.tnum`).
- **زوايا:** بطاقات `14px`، أزرار/مدخلات `8–10px`، شارات/بحث `9999px`.
- **الظل:** خفيف جداً — `0 1px 2px / 0 1px 3px rgba(16,24,40,.04–.05)`.

## 4) الخطوط
- لاتيني: **Inter**. عربي: **IBM Plex Sans Arabic**. (تُحمّل عبر Google Fonts في `[locale]/layout.tsx`، مع تدرّج احتياطي للنظام.)
- المكدّس: `var(--font-sans)`.

## 5) الصفحات المرجعية (من اللقطات)
- **Dashboard** (`IBP-2`): 4 بطاقات KPI (تنتهي/بانتظار/تجديدات/عمولات) ← «مهام عاجلة» (عمودان) + «تجديدات قادمة 12 شهراً» (رقم + رسم أعمدة) ← «النشاط الأخير».
- **Clients** (`IBP-1`): عنوان + زر «عميل جديد» ← تبويبات (الكل/منشآت/أفراد بعدّادات) + بحث ← جدول (العميل/جهة الاتصال/المدينة/وثائق فعّالة/حالة التحقق/آخر تفاعل).
- **Commissions** (`IBP-3`): عنوان + «تصدير حسب…» ← 4 بطاقات KPI ← جدول (رقم الوثيقة/العميل/شركة التأمين/النسبة/المبلغ/المستلم/الفرق/الحالة).
