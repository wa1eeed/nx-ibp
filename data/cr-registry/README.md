# مجلد لقطات السجل التجاري (Open Data — وزارة التجارة)

ضع هنا ملفّات لقطة «السجلات التجارية القائمة» بصيغة `.xlsx` (أو `.csv`) كما تُنزَّل من
[البيانات المفتوحة](https://open.data.gov.sa/ar/datasets/view/aef772cd-354c-48bb-9819-c60706dc8b56).

## المزامنة
```
node packages/db/prisma/import-cr-dir.cjs                # يستورد الجديد فقط (data/cr-registry)
node packages/db/prisma/import-cr-dir.cjs --fresh        # لقطة نظيفة: يمسح ثم يعيد الاستيراد الكامل
CR_REGISTRY_DIR=/srv/cr node packages/db/prisma/import-cr-dir.cjs   # مجلد مخصّص
```
- يستورد **الملفّات الجديدة فقط** (يتخطّى ما استُورد سابقًا عبر الاسم+الحجم+زمن التعديل).
- **إضافة شركات جديدة** = أسقِط ملفًّا جديدًا هنا ثم شغّل المزامنة (أو اتركها لمهمّة cron).
- **لقطة فصلية جديدة** = استبدل الملفّات وشغّل `--fresh`.
- يتطلّب `unzip` على النظام. البيانات نفسها لا تُرفَع على git (مُتجاهَلة).
