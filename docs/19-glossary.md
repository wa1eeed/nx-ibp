# 19 — المسرد (Glossary)

> مرجع موحّد للمصطلحات التأمينية والتقنية والتنظيمية المستخدمة في IBP، بالعربية والإنجليزية.

## مصطلحات التأمين والوساطة

| المصطلح | English | الشرح |
|---|---|---|
| وسيط | Broker | وسيط التأمين الذي يمثّل العميل أمام شركات التأمين |
| Captive / غير Captive | Captive / Non‑captive | علاقة العميل بشركة تأمين بعينها أو حرّيته في المفاضلة |
| طلب الأسعار / سلِب | Slip / RFQ | مستند يلخّص المخاطر ويُرسل لعدة شركات تأمين لطلب عروض |
| عرض سعر | Quotation | عرض شركة تأمين (قسط، تحمّل، حدود، شروط) |
| جدول المقارنة | Comparison Sheet | جدول آلي يقارن عروض الشركات (السعر/الحدود/الشروط/التحمّل) |
| أمر الإسناد | Firm Order | اختيار العميل للعرض الأنسب ⇒ المعاملة جاهزة للإصدار |
| وثيقة | Policy | عقد التأمين الصادر |
| ملحق | Endorsement | تعديل على وثيقة سارية (إضافة/حذف/تعديل/إلغاء) |
| قسط | Premium | مبلغ التأمين المستحق |
| تحمّل | Deductible | المبلغ الذي يتحمّله المؤمَّن له قبل التغطية |
| حد التغطية | Limit | الحد الأقصى لمسؤولية المؤمِّن |
| مبلغ التأمين | Sum Insured / Sum Assured | قيمة الأصل/الحياة المؤمَّن عليها |
| الاكتتاب | Underwriting | التقييم الفني للمخاطر وإعداد العروض/الوثائق |
| السجل الطبي | Census | قائمة التابعين في التأمين الطبي الجماعي |
| KYC / KYB | KYC / KYB | اعرف عميلك (أفراد) / اعرف منشأتك (شركات) |

## مصطلحات مالية وتنظيمية

| المصطلح | English | الشرح |
|---|---|---|
| الواجب الائتماني | Fiduciary Duty | التزام الوسيط بفصل أموال العملاء عن أمواله |
| داخل/خارج الميزانية | On/Off Balance Sheet | فصل أقساط العملاء (On) عن عمولات الوسيط (Off) في COA |
| شجرة الحسابات | COA (Chart of Accounts) | تصنيف الحسابات المالية (3 مستويات، 17 رقماً) |
| السندات | Vouchers | قيد يومية (JRV)، صرف (PYV)، قبض (RCV)، دفع مباشر (DPV) |
| إشعار مدين/دائن | Debit/Credit Note | مستند تسوية يُصدر للعميل |
| فاتورة ضريبية | Tax Invoice | فاتورة العمولة الصادرة لشركة التأمين (متوافقة ZATCA) |
| مركز تكلفة | Cost Center | وحدة لتوزيع الإيرادات/المصاريف (تبدأ بالفرع) |
| هيئة التأمين | Insurance Authority | المنظّم للتأمين في السعودية (تولّى مهام ساما 2023) |
| ZATCA | ZATCA | هيئة الزكاة والضريبة والجمارك — الفوترة الإلكترونية |
| PDPL | PDPL | نظام حماية البيانات الشخصية |
| NCA | NCA | الهيئة الوطنية للأمن السيبراني (ECC/CCC/DCC) |
| نفاذ/يقين/واثق/العنوان الوطني | Nafath/Yaqeen/Wathiq/SPL | موفّرو التحقّق الحكومي |

## مصطلحات تقنية

| المصطلح | English | الشرح |
|---|---|---|
| مستأجر | Tenant | شركة وساطة معزولة (بياناتها ومستخدموها وإعداداتها) |
| تعدّد المستأجرين | Multi‑tenancy | عزل كامل بين الشركات على منصة واحدة |
| Entitlement | Entitlement | صلاحية/حد ميزة ضمن باقة (INCLUDED/QUOTA/METERED/ADDON/DISABLED) |
| RBAC | RBAC | تحكّم بالوصول حسب الدور (module × CRUD) |
| ALS | AsyncLocalStorage | سياق الطلب الذي يحمل المستأجر طوال عمره |
| Middleware / Guard | Middleware / Guard | طبقات NestJS للوسيط والتفويض |
| DSL | DSL | لغة وصف النموذج (FormSchema) |
| FormSchema | FormSchema | مخطّط النموذج الديناميكي لكل فرع |
| RequestBlockRow | RequestBlockRow | مخزن عام لصفوف الكتل المتكررة لأي منتج |
| Migration / Seed | Migration / Seed | ترحيل المخطط / زرع البيانات الوهمية |
| Presigned URL | Presigned URL | رابط تخزين موقّت مشفّر قصير العمر (5 دقائق) |
| Monorepo | Monorepo | مستودع موحّد (apps + packages) بـ pnpm |
