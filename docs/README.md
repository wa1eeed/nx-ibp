## اصطلاحات التوثيق
- العربية لغة الشرح، والمصطلحات التقنية بالإنجليزية.
- كل ملف: عنوان `NN — عربي (English)`، مقدّمة، جدول محتويات، ثم محتوى بجداول/مخططات mermaid، وقسم «انظر أيضاً».
- المراجع الكودية بمسارات نسبية. **التوثيق يُحدَّث بعد كل مرحلة** (CLAUDE.md / تعريف «تمّ»).
